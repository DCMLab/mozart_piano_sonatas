# The Annotated Mozart Sonatas: Score, Harmony, and Cadence (v2.0)

Scores, chord labels and cadence labels for Mozart's 18 piano sonatas, following
the [Neue Mozart Ausgabe](https://dme.mozarteum.at/DME/nma).
